# Crypto Trading Simulator (Python)

A simple Python project that simulates basic crypto trading.

The program generates random BTC and ETH prices, allows buying and selling using a wallet, and stores transaction history in a file.

## Features
- Random price generation for BTC and ETH
- Wallet system (USD, BTC, ETH)
- Buy and sell crypto
- Transaction history saved to a file
- View and clear transaction history
